# Jogo-da-velha
Projeto de Jogo da Velha desenvolvido em HTML, CSS e JavaScript como atividade prática do curso do SENAI. O jogo possui tabuleiro 3x3 interativo, alternância entre jogadores X e O e verificação automática de vitória ou empate. O objetivo do projeto é praticar lógica de programação e manipulação do DOM no desenvolvimento web.
